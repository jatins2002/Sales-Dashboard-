# 📊 Sales & Customer Analytics Dashboard | Tableau

An interactive Tableau dashboard designed to analyze **sales performance, customer behavior, profitability, and business trends** using multiple datasets. This project provides business insights through dynamic KPIs, visualizations, and dashboard navigation.

---

## 🚀 Project Overview

This project consists of two interactive dashboards:

- 📈 **Sales Dashboard**
- 👥 **Customer Dashboard**

The dashboards help monitor key business metrics, identify sales trends, analyze customer performance, and support data-driven decision making.

---

## 📂 Dataset

The project uses the following datasets:

- Customers
- Orders
- Products
- Location

---

## 📊 Dashboard Features

### 📈 Sales Dashboard

- Total Sales KPI
- Total Profit KPI
- Total Quantity Sold
- Monthly Sales Trend
- Highest & Lowest Sales Month
- Sales vs Previous Year Comparison
- Profit vs Loss by Subcategory
- Weekly Sales & Profit Trends
- Interactive Filters

---

### 👥 Customer Dashboard

- Total Customers
- Total Orders
- Average Sales per Customer
- Customer Distribution by Number of Orders
- Top 10 Customers by Profit
- Customer Growth vs Previous Year
- Interactive Navigation
